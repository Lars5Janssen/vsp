Request Description
### DELETE component from star This endpoint allows a component to unregister itself from a star. It can also be used by SOL to remove a specific component. #### Path Variables - `COM-UUID`: COM-UUID of the component to be unregistered. #### Request Parameters - `star`: STAR-UUID of the star. #### Response The response format is `text/plain`, where the text is always empty. If the request is being sent **from a component to SOL**, the following response codes are valid: - `200`: OK: Component successfully removed from the star. - `401`: Unauthorized: Invalid IP of requesting component and/or unknown/wrong STAR-UUID. - `404`: Not Found: Component not known by SOL. If instead the request is being sent from SOL to a component, these responses are valid: - `200`: OK: Component successfully removed from the star. - `401`: Unauthorized: Invalid IP of requesting SOL and/or unknown/wrong STAR-UUID.
